# scRNA-seq Latent Subspace Recovery

Mapping latent biological subspace recovery in scRNA-seq preprocessing under sparsity, depth, and dropout stress.

## Structure
- R/01_simulation/   — Simulated dataset generation
- R/02_preprocessing/ — Preprocessing pipelines
- R/03_metrics/      — Evaluation metrics
- R/04_analysis/     — Analysis and visualization
- data/simulated/    — Simulated datasets
- data/real/         — Real datasets
- results/           — Figures and tables
